# Sample Research

This folder contains research outputs generated by the STORM Fusion Research Pipeline. Each report was produced by running the full 5-phase STORM pipeline with a 4-model Fusion panel (Owl Alpha + Nemotron 3 Ultra + gpt-oss-120b + Gemma 4 31B, fused by DeepSeek V4 Flash).

These are demonstrations of what the pipeline produces -- real research outputs from real runs, not documentation.

To reproduce these results, run the pipeline with the same topic prompt and Fusion configuration. Results will vary slightly as model versions update over time.
